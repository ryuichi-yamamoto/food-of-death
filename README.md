# README

# LethalDose(リーサルドーズ)

# 本番環境URL
https://lethal-dose.herokuapp.com/

# コンセプト
食材全てに存在する『致死量』。
身近でありながら意外と知らない、しかし生死に直結する数字。
そんな致死量をPOPに共有できる様なサービスを目指し作成しました。
より多くの方に気軽に利用して頂ける様に、ユーザー登録機能、ログイン機能は無し。

# 機能一覧
・食材の登録【食材名】【致死量】【単位】【説明】
・食材の編集【食材名】【致死量】【単位】【説明】
・食材の削除
